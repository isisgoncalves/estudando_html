Como inserir Links

Ao clicar " a ", é acionada a tag < a href=""

Responsável pelo direcionamento de links na página.

Exemplos:

Google

< a href=" https://google.com" > Google (página direcionada) < / a >

Instagram

< a href=" https://instagram.com/Usuario " > Instagram < / a >

Facebook

< a href=" http://facebook.com/Usuario " > Facebook < / a >

Fale Conosco

< a href=" mailto: usuario@provedor.com " > Fale Conosco < / a >

Abrir link em uma nova Aba

Tag target

Para que o seu link seja direcionado para uma nova Aba ao clicar, deve-se inserir a seguinte tag antes do link de acesso:
target="blank"

DOCTYPE

No HTML , o DOCTYPE é a introdução " < ! DOCTYPE html > " encontrada no topo de todos os documentos. Basta clicar a tecla ! e aparecerá a tag.

Modelo:


< ! DOCTYPE html >
< html lang="en" > (Modificar a linguagem para "pt-br")
< head >
< meta charset="UTF-8" >
< meta http-equiv="X-UA-Compatible" content="IE=edge" > (Sempre excluir esta linha, que contém EDGE no final. Esta função caíu em desuso e pode causar erros no código.)
< meta name="viewport" content="width=device-width, initial-scale=1.0" > (Nunca excluir a 3° linha, onde contem o VIEWPORT. É responsável por manter a visualização em qualquer tela, sem sofrer alteração de tamanho. Desde um smartphone, tablet, a um monitor de computador)
< title > Document (Substituir por um Título do Projeto) < / title >
< / head >
< body >
Corpo: Conteúdo do Projeto a ser executado.
< / body >
< / html > (Fim do documento)

Inserir Imagem

Tag: < img />

Na tag img devemos fecha-la com "/>"
Não há necessidade de fechar como as outras tags, pois não tem conteúdo interno.
Mas devemos inserir a localização da imagem com as seguintes informações:
Após o < img inserir src="" (source: fonte). Refere-se a origem da imagem.
Na pasta images, deve conter a imagem a ser selecionada. Importante ressaltar que ao salvar a imagem, devemos renomea-la para fácil localização e insercão no html.
Exemplo: < img src= "images/IMAGEM1.jpg" />
Este comando indica o nome da pasta (images) e o nome do arquivo a ser anexado (IMAGEM1.jpg)

Acessibilidade na imagem:

Com a função ALT após o nome da imagem, possibilita ao navegador ter uma descrição da imagem.
Em caso de erro, ou até mesmo por uma questão de acessibilidade visual.
Acrescenta-se: alt="Descrição da imagem"
Exemplo: < img src= "images/IMAGEM1.jpg" alt= "Descrição da imagem 1" />
(isto requer o máximo de detalhes na descrição)

DIV: Agrupamento/ caixinhas

Pode ser colocado texto, imagens, links ou qualquer outro conteúdo DENTRO da caixa.
As DIVs podem ser utilizadas diversas vezes em uma página, com diversos conteúdos.
Para diferencia-las, podemos usar um ID (identificador) desta forma:
Exemplo: < div id= "box1" > < / div >
Desta forma podemos estilizar cada caixinha no CSS de uma forma diferente, através de um ID (Identificador).
Abreviação de div: Inserir . (ponto) e a palavra da div a ser nomeada.
Exemplo: .teste (+ tecla ENTER)
Tag: < div class= "text" > < /div >
Automaticamente a Tag da div se inicia já nomeada.

Heading= Título

Sempre que houver o "H" (heading), refere-se a título.
A numeração diferencia apenas o tamanho da letra.
O Heading varia entre h1 a h6.

Parágrafo

Tag: < p > < /p >

O "P" refere-se a parágrafo.
Normalmente utilizado após um título.
Pode ser empregado diversas vezes dentro de um texto.

Quebra de Linha - Sigla de Break

Tag: < br >

Pode ser usado para quebra de linha em um mesmo parágrafo, sem a necessidade de criar mais um parágrafo.

Listas

Tag: < ol > < /ol >

"ol" Lista Ordenada: Uma Lista com uma organização numérica.

Tag: < ul > < /ul >

"ul" Lista Não ordenada: Uma lista organizada apenas com símbolo.

Tag: < li > < /li >

"li": Usado dentro da Tag de lista, para inserir cada item.

Botão

Tag: < button > < /button >

Button: Botão

Seção

Tag: < section > < /section >

Tag Section: É uma "seção" similar a uma "div" (caixa), mas não a substitui.
